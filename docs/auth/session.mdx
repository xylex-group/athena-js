# Auth Session

## Endpoints and methods

- `auth.getSession()` -> `GET /get-session`
- `auth.signOut()` -> `POST /sign-out`
- `auth.session.list()` -> `GET /list-sessions`
- `auth.session.revoke()` -> `POST /revoke-session` or `POST /revoke-sessions` (payload-shape dependent)
- `auth.session.revokeOther()` -> `POST /revoke-other-sessions`

## Examples

```ts
import { auth } from "./auth-client"

const session = await auth.auth.getSession()
const signOut = await auth.auth.signOut()

const sessions = await auth.auth.session.list()

// single -> /revoke-session
await auth.auth.session.revoke({ token: "session_token_1" })

// array length 1 -> /revoke-session
await auth.auth.session.revoke([{ token: "session_token_2" }])

// array length > 1 -> /revoke-sessions
await auth.auth.session.revoke([
  { token: "session_token_3" },
  { token: "session_token_4" },
])

// tokens list length > 1 -> /revoke-sessions
await auth.auth.session.revoke({ tokens: ["session_token_5", "session_token_6"] })

const revokeOther = await auth.auth.session.revokeOther()
```
